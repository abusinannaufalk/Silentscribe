# **SilentScribe – Wearable AI Communication Aid**

A wearable AI system that converts muscle movement signals into text, designed to assist speech-impaired individuals.

---

##  Overview

SilentScribe uses sensor data and machine learning to interpret hand/arm muscle patterns and translate them into readable text. This project focuses on accessibility and real-world assistive technology.

---

##  Features

* Wearable sensor integration
* Real-time muscle signal interpretation
* ML model for pattern recognition
* Text output UI
* High impact assistive solution
